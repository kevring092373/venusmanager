# Bilder für die Webseite

Lege hier deine Bilder ab. Sie werden unter dem Pfad `images/` eingebunden.

**Vorschläge für Dateinamen** (optional, dann kann die Webseite sie automatisch nutzen):

| Datei | Verwendung |
|-------|------------|
| `logo.png` oder `logo.svg` | Logo im Header |
| `hero.jpg` | Hintergrund oder Bild im Hero-Bereich (Startseite) |
| `team.jpg` | Über-uns-Seite |
| `ablauf-1.jpg`, `ablauf-2.jpg` … | Bilder für die Ablauf-Seite |

Du kannst beliebige Namen verwenden – sobald Dateien hier liegen, können wir sie in den HTML-Seiten einbinden (z. B. `<img src="images/dein-bild.jpg" alt="…">`).
